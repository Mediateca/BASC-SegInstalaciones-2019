Las llaves: casi todos los aparatos de cerrojo son operados por una llave, una combinación numérica, una tarjeta o electricidad. Existen muchos tipos de cerraduras, por lo tanto existen muchos tipos de llaves.

Las llaves deben ser marcadas con un código para identificar el correspondiente cerrojo. Una llave nunca debe ser marcada visiblemente y cuando no está en uso debe ubicarse en ganchos, en un gabinete cerrado para llaves o en una bóveda.

Las políticas deben establecer que el reportar una llave perdida, no debe resultar en una acción punitiva. Si se lleva una auditoría y un chequeo periódico de quién tiene qué llave, el control se refuerza. El cambio de cerrojos es muy aconsejable cada 8 meses y a veces a intervalos menores.

Forma parte del procedimiento de control de llaves, el resguardo de un duplicado de las llaves, en un lugar seguro y con protocolos sobre su uso y control.

Sistema de llaves maestras: las llaves maestras totales no existen, sólo se pueden hacer llaves maestras que abran una cantidad determinada de cerraduras. A esta modalidad se la denomina "amaestramiento de llaves", y a la llave que abre todas las cerraduras de una serie dada, "llave maestra". La desventaja del sistema de llaves maestras es cuando la llave maestra se pierde o es robada en este caso se compromete la seguridad de toda la empresa.

  1. Llave de cambio: una llave para una sola cerradura.
  2. Llave submaestra: abre los cerrojos de un sistema (área específica).
  3. Llave maestra: abre los cerrojos de varios sistemas.
  4. Llave gran maestra: abre todos los cerrojos (debe ser evitado).

Las llaves deben ser accesibles solamente para aquellos cuyos deberes requieren el acceso. Es por esto que debe hacerse un inventario de los sistemas de llaves, por lo menos anualmente. Cuando se pierde una llave, deben investigarse las circunstancias y dejar antecedentes escritos del caso. Al entregar una llave debe registrarse:

  1. Número de la llave
  2. Datos de la persona que la recibe
  3. Puesto o cargo
  4. Fecha de entrega
